# Spring MVC Tutorial

###  Overview

You will learn about

- Basic Application
- Model, Controllers and ViewResolver 
- Forms - DataBinding, Validation
- Annotation based approach - @RequestParam, @PathVariable, @ModelAttribute etc
- Internationalization
- Exception Handling
- Basic REST Services
